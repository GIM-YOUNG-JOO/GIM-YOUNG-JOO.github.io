# Branch
Git에서 독립적인 작업을 위한 분리 작업 공간
메인 코드와 별개로 안전하게 새로운 기능을 개발하는게 목적

- 브랜치 목록 확인
```bash
git branch
```
- 새 브랜치 생성
```bash
git branch "branch_name"
```
- 브랜치 이동
```bash
git branch "branch_name"
# or
git switch "branch_name"
```
|기능|git checkout|git switch|
|---|---|---|
|브랜치 이동|✅|✅|
|새 브랜치 생성 후 이동|✅(`-b`옵션 필요)|✅(`-c`옵션 필요)|
|특정 파일 복원|✅|❌|
|특정 커밋으로 워킹 디렉토리 복원|✅|❌|
|명령어의 명확성|복잡(다양한 기능 포함)|간단(브랜치 작업만 지원)|
